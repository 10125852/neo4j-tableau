# Instructions for Mac Installation

1. Build it:

        mvn clean package

2. Copy target/neo4j-unmanaged-extension-tableau-1.0.0.jar to the plugins/ directory of your Neo4j server.

Tableau TDE on Ubuntu machine

3) Copy jar files to plugins directory:

a) commons-lang3-3.4.jar
b) jna-3.5.1.jar
c) tableau-commmon-9.1.0.jar
d) tableau-extract-9.1.0.jar
e) tableau-server-9.1.0.jar

4) Configure the neo4j-server.properties file:
org.neo4j.server.thirdparty_jaxrs_classes=org.neo4j.unmanaged.extension.tableau=/export

5) Install the Tableau components as outlined here:
https://onlinehelp.tableau.com/current/api/sdk/en-us/help.htm#SDK/tableau_sdk_installing.htm%3FTocPath%3D_____3

6) Test the Tableau Connector.
